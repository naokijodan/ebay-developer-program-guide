デベロッパープログラムとは？

eBayが提供する無料の開発者向けプログラムです。登録すると、eBayの機能をプログラムから操作するための「APIキー」が発行されます。

APIキーがあると、たとえば次のようなことがツールから自動でできるようになります：

  • シッピングポリシーの一括作成・更新
  • 出品情報の自動送信
  • 在庫・注文データの取得
費用：プログラムへの登録は完全無料です。通常のeBayアカウントがあれば申請できます。
📋

事前に用意するもの

  • eBayセラーアカウント（すでにお持ちのもの）
  • メールアドレス（できればビジネス用。Gmailでも可）
ポイント：公式はビジネスメールアドレスを推奨していますが、GmailなどのフリーメールでもAプログラムに参加できます。
📝

登録手順（5ステップ）

1

登録ページを開く

下のボタンから登録フォームにアクセスします。

登録フォームを開く（eBay公式）
2

メールアドレスを入力する

「Email address」欄にメールアドレスを入力し、「Re-enter email address」欄にもう一度同じアドレスを入力します。

3

利用規約に同意する

「eBay API License Agreement」のチェックボックスにチェックを入れます。

規約内容は英語ですが、APIを使ってツールを作ること・安全に利用することへの同意です。
4

ロボットでないことを確認する

「I'm not a robot」のチェックボックスにチェックを入れます（reCAPTCHA）。

5

「Join」ボタンを押して送信する

フォームを送信します。約1営業日後に承認メールが届きます。

承認されると、登録したメールアドレスに通知が届きます。届いたらステップ2（APIキー取得）に進みましょう。
🔑

承認後：APIキーセットの作成

承認メールが届いたら、eBayデベロッパーサイトにサインインしてAPIキーを取得します。

1

デベロッパーサイトにサインインする

developer.ebay.com にアクセスし、登録したメールアドレスでサインインします。

2

「Application Keys」ページに移動する

サインイン後、「My Account」または「Hi Keys」から「Application Keys」ページに移動します。

3

アプリ名を入力してキーセットを作成する

アプリケーション名（任意）を入力し、「Production」の下にある「Create a keyset」ボタンをクリックします。

Sandbox と Production の違い：
・Sandbox：テスト用（実際のeBayに影響しない）
・Production：実際のeBayアカウントに接続する本番用
4

「データ保存なし」で免除申請をする

キーセット作成後、「Your Keyset is currently disabled」と表示されます。これはeBayのプライバシー規制への対応として、全開発者に義務付けられている設定です。

これは何？：eBayユーザーが「自分のデータを消してほしい」と申請したとき、eBayが開発者に通知を送る仕組みです。自前サーバーにバイヤーのデータを保存している開発者は、この通知を受け取って削除対応する必要があります。

スクリプトを使うだけの個人セラーは「免除申請」でOKです。バイヤーのデータを自分のサーバーに保存しているわけではないので、以下の手順で免除を申請します：

  1. 表示されたリンクをクリックして設定ページを開く
  2. 「Not persisting eBay data」トグルをオンにする
  3. 表示される確認ダイアログで「Confirm」をクリック
  4. 免除理由を選択して「Submit」をクリック
Submitが完了するとキーセットが有効になります。
注意：この手続きをしないとProductionキーが使えません。必ず完了させてください。
キーセットが作成されると「App ID（Client ID）」「Dev ID」「Cert ID（Client Secret）」の3つが発行されます。これをツールに設定すれば連携完了です。

よくある疑問

登録に費用はかかりますか？
完全無料です。クレジットカードの登録も不要です。
承認に何日かかりますか？
通常、申請から約1営業日（米国時間）です。土日・祝日は含まれません。
通常のeBayアカウントとは別のアカウントが必要ですか？
いいえ。デベロッパーアカウントは既存のeBayアカウントとは別の登録です。同じメールアドレスで登録できます。
App ID・Dev ID・Cert ID はどこで確認できますか？
サインイン後、「Application Keys」ページで確認できます。他人に見せないように保管してください。
APIキーを使えるのは自分のeBayアカウントだけですか？
はい。発行されたAPIキーは、登録したデベロッパーアカウントに紐づいたeBayアカウントでのみ使用できます。
🚀

登録したら何ができる？

APIキーを取得すると、eBayをプログラムから操作できるようになります。自分で一つひとつ手動でやっていた作業が、一瞬で終わるようになります。

できることの例

  • シッピングポリシーの一括作成・変更（国ごとの送料設定など）
  • 出品データの自動送信・更新
  • 在庫・注文情報の取得と管理
  • 複数アカウントの一括操作
この記事はここまで。登録〜APIキー取得が完了すれば、あとはツールやスクリプトを使って実際の操作を行います。具体的な使い方は別の資料をご確認ください。
🤖

【発展】AIに任せると、もっと先へ行ける

リテラシーの高い方向けの話です。APIキーが手元にある状態でAI（Claude Codeなど）に環境構築を任せると、さらに自動化が進みます。
